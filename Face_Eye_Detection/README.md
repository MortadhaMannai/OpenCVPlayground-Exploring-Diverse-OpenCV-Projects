# FACE AND EYE DETECTION 
This is basically an implementation of a Face and Eye Detection using Opencv Default Haar Cascade .

#### Required Modules :
  - Numpy   
    ```bash
      pip install numpy
    ```
  - Opencv   
    ```bash
      pip install opencv-python
    ```
